This project is a website that represents some of the skills I learned in my server-side scripting course, including PHP and database connections. It consists of a homepage that showcases a collection of images of my paintings, a contact page to submit a message, and an account page that allows users to create an account and then log in with those same credentials to view a screen that can only be accessed when logged in.
