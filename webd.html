encycloPETia

user-centered design project
2021

This project was a semester-long group production of the user experience design process of an app called encycloPETia, that I completed during my user-centered design course. This plan was built from thorough research and planning and consists of a site map, an evaluation, and critique of marketplace examples, defined users and stakeholders, market survey data, user goals, tasks, and activities, use case scenario with think-aloud protocol, tasks analysis, interaction analysis, and design evaluation, a data dictionary, information architecture, navigation, search, sort, and retrieval results, low fidelity prototype, wireframe, and various high-fidelity designs.

j.art

server-side scripting project
2020

This project is a website that represents some of the skills I learned in my server-side scripting course, including PHP and database connections. It consists of a homepage that showcases a collection of images of my paintings, a contact page to submit a message, and an account page that allows users to create an account and then log in with those same credentials to view a screen that can only be accessed when logged in.

exercise app

mobile development project
2021

This project is a React app that is a combination of some of the skills I learned during my mobile development course. It lists 7 different exercises that can either be timed or counted. Exercises that have been completed can be checked off, and the total number of time spent exercising can be entered at the bottom.
